# 🎓 Student Registration Form

A simple and responsive **Student Registration Form** built using **HTML5**. This project demonstrates the use of form elements, input validation, semantic HTML structure, and table creation to display student information.

---

## 🚀 Features

✅ Student Registration Form

✅ HTML5 Form Validation

✅ Required Input Fields

✅ Radio Buttons for Gender Selection

✅ Checkboxes for Skills Selection

✅ Date of Birth Picker

✅ Submit Button

✅ Student Data Displayed in Table Format

✅ Semantic HTML Structure

---

## 🏗️ Technologies Used

* HTML5
* Semantic HTML
* Form Validation
* Tables
